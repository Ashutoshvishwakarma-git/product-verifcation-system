# product-verifcation-system
A full-stack MERN web application designed to help users verify the authenticity of products using a unique product code. This project aims to reduce counterfeit products in the market by providing a trustworthy verification platform for both consumers and brands.  Built with modern technologies like React, Node.js, Express, and MongoDB Atlas.
 Key Highlights
  User Authentication: Secure signup and login with encrypted passwords (JWT)
  Product Verification: Instantly verify product authenticity using a unique product code
  Admin Controls: Add or manage product details (optional role-based access)
  Responsive Design: Built using Tailwind CSS for perfect viewing on all devices
  Cloud Database: Integrated with MongoDB Atlas for scalable storage
  Deployed on Render: Both backend and frontend fully deployed online
